## 安装
- git clone git@github.com:xun-zi/my-music.git
- npm install
- 克隆 NeteaseCloudMusicApi
- [NeteaseCloudMusicApi地址](https://github.com/Binaryify/NeteaseCloudMusicApi)
- cd NeteaseCloudMusicApi
- pnpm i
